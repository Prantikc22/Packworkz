ID GENERATION FUNCTION
Create a Supabase edge function 
generate_id(prefix, table, column):
  year = current year
  last_id = SELECT MAX(column) 
            FROM table 
            WHERE column LIKE 
            prefix-year-%
  sequence = last_id ? 
    parseInt(last_id.split('-')[2]) + 1 : 
    10001
  return prefix-year-sequence

Use for all IDs:
PKG (quote_requests.quote_id)
ORD (orders.order_id)
DES (design_requests.design_id)
SAM (sample_requests.sample_id)
INV (invoices.invoice_id)

vendor_count = '1' | '2-3' | '5+'

BUSINESS LOGIC FUNCTIONS

Upfront discount calculation:
function calculateUpfrontDiscount(
  price: number, 
  paymentType: string
): number {
  if (paymentType === 'upfront') {
    return Math.round(price * 0.03)
  }
  return 0
}

Credit eligibility check:
Trigger on orders.status update to 
'delivered':
  UPDATE users 
  SET orders_completed = 
    orders_completed + 1
  WHERE id = order.user_id
  
  IF orders_completed >= 3:
    UPDATE users 
    SET credit_eligible = true
    WHERE id = order.user_id

Sample fee adjustment:
In admin order creation:
  SELECT * FROM sample_requests
  WHERE user_id = ? 
  AND product_id = ?
  AND status = 'delivered'
  AND adjusted_to_order_id IS NULL
  
  If found AND order_total > 50000:
    Show admin alert:
    "Customer has a delivered 
    sample (SAM-XXXXX) for 
    this product. 
    Adjust ₹[amount] from order?"
    On confirm:
      Deduct from invoice amount
      UPDATE sample_requests 
      SET adjusted_to_order_id = 
        new_order.id
