STEP 4: FLOATING ANIMATIONS

Each card gets a unique 
floating animation — 
slow, smooth, organic.

@keyframes float-1 {
  0%, 100% { transform: translateY(0px); }
  50% { transform: translateY(-10px); }
}
@keyframes float-2 {
  0%, 100% { transform: translateY(0px); }
  50% { transform: translateY(-14px); }
}
@keyframes float-3 {
  0%, 100% { transform: translateY(0px); }
  50% { transform: translateY(-8px); }
}
@keyframes float-4 {
  0%, 100% { transform: translateY(0px); }
  50% { transform: translateY(-12px); }
}
@keyframes float-5 {
  0%, 100% { transform: translateY(0px); }
  50% { transform: translateY(-9px); }
}
@keyframes float-6 {
  0%, 100% { transform: translateY(0px); }
  50% { transform: translateY(-11px); }
}

Assign to cards:
Card 1: animation: float-1 6s ease-in-out infinite
Card 2: animation: float-2 7s ease-in-out infinite; animation-delay: -2s
Card 3: animation: float-3 5.5s ease-in-out infinite; animation-delay: -1s
Card 4: animation: float-4 6.5s ease-in-out infinite; animation-delay: -3s
Card 5: animation: float-5 7.5s ease-in-out infinite; animation-delay: -1.5s
Card 6: animation: float-6 6s ease-in-out infinite; animation-delay: -4s

The negative animation-delays 
start each card mid-animation 
so they never float in sync — 
looks completely organic.

On hover: 
pause animation:
animation-play-state: paused

STEP 5: ENTRANCE ANIMATION

When page loads, 
cards animate in from below:

Each card starts:
opacity: 0
transform: translateY(30px)

Then transitions to:
opacity: 1
transform: translateY(0)
transition: opacity 0.6s ease, 
            transform 0.6s ease

Stagger the delays:
Card 1: transition-delay: 0.3s
Card 2: transition-delay: 0.5s
Card 3: transition-delay: 0.4s
Card 4: transition-delay: 0.7s
Card 5: transition-delay: 0.6s
Card 6: transition-delay: 0.8s

Trigger by adding a class 
'loaded' to the container 
after component mounts:

useEffect(() => {
  setTimeout(() => {
    container.classList.add('loaded')
  }, 100)
}, [])

.loaded .card {
  opacity: 1;
  transform: translateY(0);
}
